OpenSource Groupware
=========

OpenSource Groupware Develop Project

  - 오픈소스로 제공하는 그룹웨어 개발 프로젝트
  - 중/소기업을 타겟으로 저비용 고효율 그룹웨어를 개발
  - 멀티플랫폼 서비스로 사용 가능한 플랫폼을 확대
